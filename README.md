# 장르별 온라인 이용건수 기반 영화 선호도 분석

## 프로젝트 소개

이 프로젝트는 장르별 온라인 이용건수 엑셀 데이터를 활용하여 영화 및 영상 콘텐츠의 장르별 선호도를 분석하는 Streamlit 프로젝트입니다.

온라인 이용건수가 높다는 것은 해당 장르가 온라인에서 많이 소비되었다는 뜻으로 볼 수 있습니다. 따라서 본 프로젝트에서는 온라인 이용건수를 장르 선호도를 판단하는 주요 기준으로 사용했습니다.

## 디렉터리 구조

```text
pages/1_문제정의및데이터수집.py
pages/2_데이터전처리.py
pages/3_데이터시각화.py
pages/4_인공지능모델링.py
README.md
main.py
장르별_온라인이용건수_20260707_023139.xlsx
requirements.txt
```

## 사용 데이터

사용 파일은 `장르별_온라인이용건수_20260707_023139.xlsx`입니다.

주요 컬럼은 다음과 같습니다.

- 순위
- 장르
- 작품편수
- 온라인 이용건수
- 점유율(%)

## 사용한 기술

- Python
- Streamlit
- pandas
- plotly
- scikit-learn
- openpyxl

## 페이지 구성

### 1. 문제정의 및 데이터수집

프로젝트 주제, 분석 목적, 사용 데이터, 핵심 질문을 설명합니다.

### 2. 데이터전처리

엑셀 파일을 불러온 뒤 분석에 필요한 형태로 정리합니다.
숫자 변환, 빈 행 제거, 합계 행 제거, 파생 변수 생성을 수행합니다.

### 3. 데이터시각화

장르별 온라인 이용건수, 이용점유율, 작품당 이용건수를 그래프로 나타냅니다.

### 4. 인공지능모델링

의사결정나무 모델을 사용하여 장르의 선호등급을 간단히 분류합니다.
데이터 수가 적기 때문에 실제 서비스용 모델이 아니라 수행평가용 학습 예시입니다.

## 실행 방법

필요한 라이브러리를 설치합니다.

```bash
pip install -r requirements.txt
```

Streamlit 앱을 실행합니다.

```bash
streamlit run main.py
```

## 분석 기준

본 프로젝트에서는 온라인 이용건수를 장르 선호도 판단 기준으로 사용했습니다.
온라인 이용건수가 많은 장르는 사람들이 온라인에서 많이 이용한 장르로 해석했습니다.

다만 이 데이터는 개별 사용자의 실제 취향 설문이 아니라 장르별 집계 데이터입니다. 따라서 분석 결과는 개인별 취향 분석이 아니라 장르별 온라인 이용 경향 분석으로 해석해야 합니다.
